== Description ==

The easiest way to create beautiful charts. 
Create your charts with a easy interface and have full control over your chart.
 
== Installation ==
 
1. Put the plugin folder into [wordpress_dir]/wp-content/plugins/
2. Go into the WordPress admin interface and activate the plugin
3. Create you charts in the "Media Insert" section above the editor or use the custom post type "Charts"
